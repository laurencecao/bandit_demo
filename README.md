# Epsilon Greedy 多臂老虎机学习演示

一个交互式网页应用，用于直观理解 **Epsilon Greedy 算法**如何在多臂老虎机问题中平衡探索与利用。

## 它演示什么

多臂老虎机问题：面前有若干老虎机（臂），每台有未知的中奖概率。你有限次机会拉杆，目标是最大化总收益——该选哪台？

核心矛盾：

- **利用（Exploit）**：选当前看起来最好的臂，稳拿已知收益
- **探索（Explore）**：试试其他臂，可能发现更好的选择

Epsilon Greedy 是最简单的策略：以概率 ε 随机探索，以概率 1-ε 选择当前最优。本应用让你亲眼看到这个策略如何逐步逼近每台机器的真实概率。

## 快速开始

```bash
pip install flask
python app.py
```

浏览器打开 `http://localhost:15000`。

## 使用方式

1. **配置参数**：设置臂数量（2~10）、每臂真实概率、ε 值、演示速度
2. **运行演示**：点击「开始」自动播放，或「单步」逐步观察
3. **观察学习过程**：蓝色进度条（估计概率）逐步逼近灰色虚线（真实概率）
4. **理解决策**：底部详情区显示每步的探索/利用选择和概率更新

## 界面说明

| 区域 | 内容 |
|------|------|
| 配置面板 | 臂数量、真实概率、ε 值、速度、控制按钮 |
| 状态面板 | 步数、探索/利用次数、累积奖励、当前最佳臂 |
| 臂卡片 | 估计概率 vs 真实概率对比条、尝试与奖励统计 |
| 历史轨迹 | 每步选择的颜色编码（橙=探索，蓝=利用，绿光=获奖） |
| 决策详情 | 每步的完整决策过程文字描述 |

## 算法逻辑

每一步：

1. 以 ε 概率随机选臂（探索），以 1-ε 概率选当前估计最优臂（利用）
2. 按该臂真实概率获得奖励（1 或 0）
3. 更新该臂的估计概率：Q(a) = (Q(a) × N + reward) / (N+1)

当所有臂的估计概率与真实概率误差均 < 0.05 时，提示算法收敛。
